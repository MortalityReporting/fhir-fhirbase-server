About This Server

This server provides an OMOP on FHIR Specification using an open source software stack.

This server is built from a number of modules of the HAPI FHIR project and OMOP on FHIR project, which are both 100% open-source (Apache 2.0 Licensed) Java based implementation of the FHIR specification.
